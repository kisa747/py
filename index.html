windows10安装方法

发表于 2019-03-27 分类于 windows

准备工具

  • windows 10的iso镜像，推荐使用 lstsc 长期支持版：cn_windows_10_enterprise_ltsc_2019_x64_dvd_9c09ff24.iso

    记得校验一下 SHA1 值：24b59706d5eded392423936c82ba5a83596b50cc

  • 一个U盘，存在一个不小于 8G 的 FAT32 格式的分区，分区表使用 GPT、MBR 无所谓。

安装步骤

  1. 做好电脑的备份工作，将电脑设置为 UEFI 启动。
  2. 将 iso 文件 解压至指定 U 盘的根目录。
  3. 使用 U 盘启动电脑，安装 windows 10。

搞定！
