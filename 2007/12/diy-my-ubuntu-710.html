DIY 属于自己的 Ubuntu 7.10

发表于 2007-12-18

默认安装的ubuntu过于臃肿，这次就先只安装个command-line system，再DIY。下面我们就开始吧。
1,用Alternate install CD启动系统，一定要 F2 设置语言为简体中文，否则安装后还得设置中文locale，安装菜单中选择”Install a command-line system“，然后跟正常安装一样。
2,安装后得到一个只有命令行的系统，确保能正常上网。
sudo nano /etc/network/interfaces

修改为以下内容

auto eth0
iface eth0 inet dhcp

#获取IP地址
sudo dhclient eth0
#用restart竟然不行，但先stop再start却可以
sudo /etc/init.d/networking stop && sudo /etc/init.d/networking start

接下来安装最基本的X视窗组间。
#设置apt源
wget http://kisa747.googlepages.com/sources.list.tar.gz
tar xvfz sources.list.tar.gz
sudo mv /etc/apt/sources.list /etc/apt/sources.list.bak
sudo cp sources.list /etc/apt/sources.list
#改回sources.lst的默认读写权限
sudo chmod 644 /etc/apt/sources.list
#系统升级
sudo aptitude update && sudo aptitude safe-upgrade
#安装X窗口，gdm，gnome核心组件
sudo aptitude install xserver-xorg-core gdm gnome-core compiz-core

注：
以下是sources.list的内容
deb http://ubuntu.cn99.com/ubuntu/ gutsy main restricted universe multiverse deb http://ubuntu.cn99.com/ubuntu/ gutsy-security main restricted universe multiverse deb http://ubuntu.cn99.com/ubuntu/ gutsy-updates main restricted universe multiverse deb http://ubuntu.cn99.com/ubuntu/ gutsy-proposed main restricted universe multiverse deb http://ubuntu.cn99.com/ubuntu/ gutsy-backports main restricted universe multiverse deb-src http://ubuntu.cn99.com/ubuntu/ gutsy main restricted universe multiverse deb-src http://ubuntu.cn99.com/ubuntu/ gutsy-security main restricted universe multiverse deb-src http://ubuntu.cn99.com/ubuntu/ gutsy-updates main restricted universe multiverse deb-src http://ubuntu.cn99.com/ubuntu/ gutsy-proposed main restricted universe multiverse deb-src http://ubuntu.cn99.com/ubuntu/ gutsy-backports main restricted universe multiverse deb http://ubuntu.cn99.com/ubuntu-cn/ gutsy main restricted universe multiverse

2,这样下来基本的X窗口已经可以了启动了，执行 startx 命令进入桌面。接着就可以在X视窗下的终端执行以下命令了：
startx
#接着就可以在X视窗下的终端执行以下命令了。
#安装apt的gui前端 新立得 和 fcitx中文输入法 ，并将fcitx设置为默认输入法。
#fcitx设置参见我的另一篇文章 我的Fcitx美化方案
sudo aptitude install synaptic fcitx
sudo im-switch -s fcitx -z default
im-switch -s fcitx -z default
#影音方案，exaile会自动把需要的gstreamer依赖包装上，这样直接就能播放mp3了
#更换了音乐播放器──Exaile 这里有关于exaile音乐播放器的简单介绍
#再装个gstreamer0.10-ffmpeg就连wma也能播放了，而且也让totem能播放wmv，dvd
#我电脑上的影音文件格式只有mp3，wma，rmvb，rm，wmv，只要能播放这几种种媒体格式就可以了
#现在才发现Ubuntu下竟一条命令就解决了我所有的影音播放。apt就是牛！
sudo aptitude install exaile smplayer w32codecs gstreamer0.10-ffmpeg
#firefox中文包
sudo aptitude install mozilla-firefox-locale-zh-cn
#受限驱动管理器，用它图形安装显卡驱动
#我的ati显卡也可以用 sudo aptitude install xorg-driver-fglrx
sudo aptitude install restricted-manager
#自动挂载U盘等可移动介质管理工具，但会自动安装totem，gthumb等好多软件，不过有了它挺方便的
sudo aptitude install gnome-volume-manager
#编辑menu bar的小工具
sudo aptitude install alacarte
#pdf阅读器+中文支持
sudo aptitude install evince xpdf-chinese-simplified
#归档文件管理器 + rar/7z 支持
sudo aptitude install file-roller rar p7zip
#类似windows记事本的文本编辑器
sudo aptitude install leafpad
#聊天工具，基于GTK
#eva基于qt，lumaqq要java环境，所以就干脆用pidgin
sudo aptitude install pidgin
#包含屏幕截图等一些工具
sudo aptitude install gnome-utils
#美观一点（包含clearlook在内几个主题），主要是它还能自动安装gtk包
sudo aptitude install gnome-themes
#基于文本的设置服务的工具
#使用及服务配置请参考 ubuntu简单优化
sudo aptitude install sysv-rc-conf
#就是ubuntu下那个添加删除程序
sudo aptitude install gnome-app-install
#图形化的安装deb包的工具，其实用dpkg命令就足够了
sudo aptitude install gdebi
#系统监视器
sudo aptitude install gnome-system-monitor
#编辑GConf配置数据库的图形界面编辑器
#设置参考 gconf-editor 的设置
sudo aptitude install gconf-editor
#星际译王+简明英汉词库
sudo aptitude install stardict-cdict-gb stardict
#nautilus右键扩展
#添加“以管理员打开”+“Open in terminal“
#用脚本也可以做到 nautilus-scripts.tar.gz
#下载解压拷到 ~/.gnome2/nautilus-scripts，然后在此目录下运行命令 chmod u+x *
sudo aptitude install nautilus-gksu nautilus-open-terminal

3,美化grub

wget http://kisa747.googlepages.com/grubuntu.xpm.gz
sudo cp ./grubuntu.xpm.gz /boot
sudo cp /boot/grub/menu.lst /boot/grub/menu.lst.bak
sudo gedit /boot/grub/menu.lst

在”timeout”项下面添加下面一段，其中 (hd0,7) 为对应的 /boot 所对应的分区

splashimage=(hd0,7)/boot/grubuntu.xpm.gz

4,一个mac风格的的鼠标样式主题，解压后拷到 ~/.icons 文件夹下就能使用了

wget http://kisa747.googlepages.com/aquadiz.tar.gz

下面就是截图中浅绿色的边框主题，解压后拷到 ~/.themes 文件夹下就能使用了

wget http://kisa747.googlepages.com/Clearlooks-green.tar.gz

一个类似windows的边框主题 zune ，解压后拷到 ~/.themes 文件夹下就能使用了

wget http://kisa747.googlepages.com/zune.tar.gz

把雅黑字体拷到 ~/icons 文件夹下，运行命令 fc-cache -f 刷新下字体缓存，然后不用任何设置现在字体就十分漂亮！
